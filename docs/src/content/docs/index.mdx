SourceBans++ is a free, open-source admin and ban-management system
for Source-engine games (Team Fortress 2, Counter-Strike, Garry's Mod,
and friends). It has two halves that talk to each other:

- A **web panel** your admins use to manage bans, mutes, server access,
  and admin permissions.
- A set of **SourceMod plugins** running on your game servers that
  enforce those bans in-game and report new admin actions back to the
  panel.

You install both, point them at the same database, and the rest is
day-to-day moderation work.
